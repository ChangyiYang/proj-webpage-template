Overview

In this project, we implement the ray tracing algorithm. First, we implement the ray generation and how ray intersect with triangle and sphere. Then we implement the BVH acceleration architecture. Next, we do Direct Illumination and improve it to Global illumination to render light correctly. In the end, we use adaptive sampling to accelerate our sampling process.

Part 1

In this part, we implement the ray generation and ray intersection with triangles and spheres.

To generate rays from a point in the image space, first, we need to translate the point to the camera space. Then use the c2w, camera-to-world rotation matrix to transfer the point from camera space to world space. In the end, we set the ray’s position to be the camera position in the world space.

Then we implement the sample method, which sample a image pixel by a uniform sampler. We generate rays to get the sample from the sample position.

For triangle intersection, I use the Moller Trumbore Algorithm to get the t,b1,b2 values for the intersection point. Then we check if the t is between min_t and max_t and whether the intersection is in the triangle by checking the b1 and b2.

For sphere intersection, we first get the unary quadratic equation for t. If the equation has no real root, then there is no intersection. If there is two real roots, we first check if the smaller root is valid then the larger root.

Part 2

To construct BVH, first, we generate the bounding box for the new node, which is needed for both leaf and interior nodes. Then, we check the number of primitives in the node. If it is less than the max_leaf_node , make this node a leaf node by assigning its start and end to the input’s start and end. If it is a interior node, we need to find which axis to split and where to split. For me, first, I calculate the variance along three axis and pick the axis with the largest variance. Then in that axis, we split over the mean of all centroids by partition() function.

Part 3

Implementing the zero-bounce illumination is quite simple. If the ray does not intersect, do nothing. If it does, check the emission of the intersect point. For one-bounce illumination, we get our samples from our sample space (the whole hemisphere or only the light ) and use reflection equation to calculate the reflection radiance.

Part 4

To implement indirect lighting function, first, we add one_bounce_radiance. Then, we sample in the hit point. From the sample, we shoot a ray to see if it intersect with any object. If not, simply return. If it does, we recursively call the at_least_one_bounce_radiance at the hit point to generate its incoming radiance. During the process, we keep track of the depth of the ray to only allow max_ray_depth bounce. Also, we use Russian roulette to randomly terminate ray tracing.

Part 5

Adaptive sampling is a method to accelerate the rendering or sampling process. It detects whether this pixel is converged. As instructed, we keep s1 (the sum of all the sample’s illumination) and s2 (the sum of all the sample’s illumination’s square). Then we use them to calculate the mean and variance of current sample. When the condition satisfied, we break the sample loop and return the result.
